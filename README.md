# CSIS Format

> Common Student Information Structure 通用学生信息结构

CSIS 是一个基于 JSON 的 DSL，旨在为随机抽名、排位、积分等教学适用的与学生数据有交集的软件提供一个统一的交换格式。

CSIS 分为多个子语言，每个子语言都适用一类特定软件。它还提供一个通用格式，便于识别。

## CSIS Unified File

- 后缀名 | CSIS

对于以下子语言全部适配。

## Common Student List Structure - 通用学生列表结构 CSLS

- 后缀名 | CSLS
- 适用场景
  - 随机抽名软件
  - 积分软件
  - 值日表软件
- 特性
  - 存储多个班级
  - 小组信息
  - 性别包容
  - 对于每个学生的额外信息
